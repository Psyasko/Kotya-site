# Kotya Site — Asian Dragon Luxe

Односторінковий Vite + React сайт-візитівка для масажиста, який працює в клініці.

## Що в цій версії

- Перероблений hero-блок: менший стартовий відступ, щоб на першому екрані одразу було видно більше інформації.
- Більше азійської стилістики: фонові контури драконів, змієподібні золоті лінії, хмарні завитки, каліграфічні вигини навколо фото.
- Виправлені органічні плитки: декоративна форма тепер окремо, а текст лежить у безпечній контентній зоні.
- Збережений темний ботанічний стиль із листям та смарагдовою палітрою.
- Оптимізація: WebP для фото й фону, менше важких blur/filter, спрощені анімації на мобільних, reduced-motion режим.
